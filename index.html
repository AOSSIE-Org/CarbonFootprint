Carbon Footprint

A browser extension that displays carbon footprint information within various map services.

View on GitHub Download .zip Download .tar.gz

Installation

The Carbon Footprint extension is available for Google Chrome, Safari and Firefox.

To install, visit Carbon Footprint's page in your browser's extension/add-on gallery/store:

  • Google Chrome Web Store
  • Safari (Coming soon...)
  • Firefox Add-On Gallery

Authors and Contributors

The Carbon Footprint extension was created by Bruno Woltzenlogel Paleo (@Ceilican) in 2011. In 2016, Chirac Arora (@heychirag), Kolya Opahle (@Kolpa) and Prateek Gupta (@PrateekGupta1509) joined the project as Google Summer of Code students.
